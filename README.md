A personal learning repo for practicing petroleum data engineering using Equinor's open **Volve
field dataset**, inspired by the **Orkahub Energy** YouTube tutorials.

## Goal

This repo is not a production tool — it's where I work through real subsurface/drilling data
end to end (PDF reports, messy real-world formatting, coordinate systems, trajectory math) to
build practical data skills, following along with:

- **Dataset:** [Volve field data set](https://www.equinor.com/energy/volve-data-sharing) — Equinor's
  full open release of subsurface and production data from the Volve field (North Sea), shared for
  research, study, and academic use.
- **Tutorials:** [Orkahub Energy YouTube playlist](https://www.youtube.com/watch?v=suenCjNsVWQ&list=PL_pV2V5A8D8GA7khlrQ1m1MVZqpCyrNi5) — walkthroughs of exploring and processing this dataset in Python.

More notebooks will be added as I work through more of the dataset and the tutorial series.

The code in this repository extends beyond the original tutorials, incorporating deeper analysis wherever possible.

## Data access

The PDF reports and other Volve files used in these notebooks aren't included in this repo — they're
large and covered by Equinor's Open Data Licence. Download them directly from
[Equinor's Volve data page](https://www.equinor.com/energy/volve-data-sharing).
